# Logger

An Node js logger and express middleware

## Installation

### npm

```bash
npm install @rohit/logger

```

### pnpm

```bash
pnpm add @rohit/logger

```

### Yarn

```bash
pnpm add @rohit/logger

```

## Quick Start

### Common js

```js
const { Logger } = require("@rohit/logger");

const logger = new Logger();

logger.info("Hello World!");
```

### Es6

```js
import { Logger } from "@rohit/logger";

const logger = new Logger();

logger.info("Hello World!");
```
